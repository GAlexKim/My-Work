Image Resize Application
G.Alex Kim

The Image_Resize_Application.py is the source file and the Image_Resize_Application.exe
is the application

How to Use:
1. Click Image_resize_Application.exe
2. Enter the full path of your image file 
Example: C:\Users\geonw\Desktop\example.jpg
3. Enter the desired width of your image
4. The new image would have been generated as a square 
image and saved on the directory of the application

Note: Please enter a correct full path as new image will not be generated if full path of the file is not entered.
